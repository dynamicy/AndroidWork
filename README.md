# AndroidWork

[![Build Status](https://travis-ci.org/dynamicy/AndroidWork.svg?branch=master)](https://travis-ci.org/dynamicy/AndroidWork.svg?branch=master)

Android clients.

## Features

* AsyncTask
* Encryption(AES+Bolofish)
* Handler + Message Passing
* Json Helper
* Android NDS
* SharedPreference
* RecyclerView
* SQLiteHelper
* Android Service
* Glide
* TimeZone(Joda-Time)
* ViewPager + activities
